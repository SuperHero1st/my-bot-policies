Privacy Policy

This document outlines the privacy policy for the Telegram bot designed to fetch and display public posts from a specific Facebook Page.

Information We Collect

This bot interacts with publicly available information. It does not collect personal information from users of the Telegram bot beyond what is inherently provided by Telegram (e.g., user ID for communication).

  • Facebook Page Data: The bot fetches publicly available posts (text, images, links) from a designated public Facebook Page. It does not access private Facebook data or require Facebook login from users.
  • Telegram User Data: To function, the bot receives messages from Telegram users and responds. Telegram user IDs may be processed to deliver messages.

How We Use Information

The information is used solely for the purpose of:

  • Retrieving public posts from the specified Facebook Page.
  • Formatting and sending these posts as messages to users of the Telegram bot.

Data Storage

The bot may temporarily store identifiers of the last post fetched from the Facebook Page to avoid sending duplicate content. No other personal data from Facebook or long-term storage of Telegram user interaction data is planned beyond what is necessary for the bot's immediate operation.

Data Sharing

We do not share any collected information with third parties, except as required to operate the bot (e.g., interacting with Telegram and Facebook APIs as per their terms).

Your Choices

You can stop using the Telegram bot at any time to cease receiving updates.

Contact Us

If you have any questions about this Privacy Policy, please create an issue on the relevant GitHub repository (if applicable) or contact the bot administrator if contact information is provided within the bot.
